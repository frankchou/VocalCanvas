## 1. 前言：策略定調

### 1.1 創辦人的兩波框架

創辦人定義了三層產品價值：功能性 → 情緒 → 資產，並決定先從**情緒價值**啟動市場。本方案尊重這個框架，同時融合市場分析師的洞察——將「情緒價值」的溝通方式從純粹的「療癒感受」調整為**「創作者用聲音說故事」**，讓情緒訴求同時具備功能性的說服力。

### 1.2 核心行銷洞察

根據競品分析與市場分析的結論：

| 洞察 | 行銷意涵 |
|------|---------|
| 情境模式是全球首創 | 這是最強的差異化武器，所有通路都要把它講到位 |
| BGM 即時混音極度稀缺 | 不只是功能，而是「完成品體驗」——別人給你半成品，我們給你成品 |
| 台灣繁中市場空白 | 不需要打敗 ElevenLabs，只需要成為「台灣人的第一選擇」 |
| 情緒價值付費轉換率較低 | 第一波需要把情緒訴求包裝成「創作工具」，而非「冥想 App」 |
| 短影音創作者是最大潛力 TA | 即使第一波主打情緒，仍須在內容中埋入創作者的使用情境 |

### 1.3 行銷哲學：一句話

> **不賣工具，賣一個「用聲音說故事」的可能性。**

## 2. 品牌定位建議

### 2.1 定位語

**VocalCanvas 不是一個 TTS 工具，是你的「聲音創作夥伴」。**

具體來說：

- 對外溝通時，避免使用「文字轉語音工具」這種技術性描述
- 改用「用 AI 把文字變成有溫度的聲音」（直接取自品牌設計系統的 brand essence）
- 產品類別定義為：**AI 聲音創作平台**

### 2.2 品牌語調（Tone of Voice）

根據品牌設計系統的內容基礎，行銷端的語調規範如下：

| 維度 | 是 | 不是 |
|------|-----|------|
| 語氣 | 溫暖、自信、像一個懂你的朋友 | 冰冷、技術控、像在讀使用說明 |
| 用詞 | 日常口語、有畫面感的動詞 | 行銷八股（「賦能」「顛覆」「一站式解決方案」） |
| 節奏 | 短句為主，偶爾一句長句建立情緒 | 堆砌形容詞的長段落 |
| 情感 | 安靜的自信——我們知道聲音可以做到什麼 | 過度興奮的推銷感 |
| 標點 | 句號收尾，不濫用驚嘆號 | 每句話都加驚嘆號或表情符號 |

**行銷端特許：** 品牌設計系統規定「產品 UI 不用表情符號」，但在社群貼文與行銷 email 中可以少量使用。Landing page 介於兩者之間，盡量不用。

### 2.3 與競品的差異化——一句話說清楚

> **其他工具讓你「合成語音」，VocalCanvas 讓你「創作有場景、有配樂的聲音作品」。**

這句話的核心邏輯：

- ElevenLabs：語音品質最強，但你拿到的是一段裸音檔，BGM 自己找、自己混
- Murf / Play.ht：功能多，但介面複雜，沒有「情境」的概念
- VocalCanvas：選一個情境，30 秒拿到一個「帶配樂的完成品」

### 2.4 品牌關鍵字雲

行銷素材中應反覆出現的詞彙群：

```
聲音 / 故事 / 創作 / 溫度 / 場景 / 呼吸 / 陪伴
作品（不說「音檔」）/ 畫布（呼應 Canvas）/ 調色盤
```

應避免的詞彙：

```
人工智慧語音合成 / 自動化配音 / 高效產出 / 企業級 / 一站式
```

## 3. 第一波行銷方案 -- 情緒價值

### 3.1 目標：建立品牌認知，切入療癒內容市場

**核心訊息：** 你的聲音，可以是一份禮物。

第一波不追求大量付費轉換，而是追求：
1. 品牌辨識度——讓目標族群知道「有這個東西存在」
2. 情緒連結——讓他們感受到「這跟其他冷冰冰的 TTS 不一樣」
3. 種子用戶——累積第一批 500-1,000 名註冊用戶

#### Slogan 與主訊息

**主 Slogan：**

> **讓文字呼吸 -- Let Your Words Breathe.**

（取自品牌設計系統中已定義的行銷語句「讓文字呼吸 -- Let your words breathe.」）

**副訊息線（依情境切換）：**

|------|--------|
| 冥想引導 | 你寫的每一句話，都值得被溫柔地說出來。 |
| 睡前故事 | 今晚，讓一個聲音陪你入睡。 |
| 正念肯定語 | 聽見自己寫下的力量。 |
| 通用 | 30 秒，從文字到有溫度的聲音。 |

#### 推廣通路

```
優先度排序：

[最高] IG / Threads
  ├── 療癒圖文社群滲透
  ├── Reels 短影音（產品 demo + 情緒氛圍）
  └── Threads 短文互動（引發共鳴的文字內容）

[高]   KOL / 微型創作者合作
  ├── 療癒圖文帳號（5K-50K 粉絲）
  ├── 冥想 / 瑜珈老師的個人帳號
  └── 心理健康科普帳號

[中]   Facebook 社團
  ├── 身心靈社團
  ├── 冥想練習社團
  └── 手帳 / 書寫療癒社團

[中]   Podcast 廣告 / 合作
  ├── 療癒類 Podcast 節目贊助
  └── 心理健康類 Podcast 來賓分享

[低]   付費廣告（Phase 1 先不投）
  └── 等轉換漏斗跑通後再投放 IG/FB 廣告
```

#### 內容策略

**能打動他們的內容類型：**

1. **「聽起來像什麼」的展示** -- 不講功能，直接放一段成品。15 秒的冥想引導片段，讓人馬上感受到「原來 AI 的聲音可以這麼溫柔」。

2. **「為自己寫一段話」的互動** -- 在社群上發起互動：「如果你可以對三天前的自己說一句話，你會說什麼？」收集回覆後，選幾則用 VocalCanvas 製作成有配樂的音檔，回覆給留言者。

3. **「睡前 60 秒」系列** -- 每週固定一則 IG Reels / Threads 貼文，用 VocalCanvas 製作 60 秒的睡前朗讀（冥想引導 / 短詩 / 肯定語），建立「這個帳號 = 睡前陪伴」的聯想。

4. **創作者幕後故事** -- 拍 Persona A 類型的創作者如何用 VocalCanvas 把圖文作品變成有聲作品的過程。重點不是功能教學，而是「她為什麼要做這件事」。

- 「你有 10 個 Podcast 的想法，卡在第 0 集。VocalCanvas 讓你從第 1 集開始。」
- 「配音員一次 NT$3,000。VocalCanvas 一個月 NT$299，無限次。」
- 「你的有聲書企劃案上面寫著『待確認配音預算』。現在可以把那行刪掉了。」

## 7. 獨立建議

### 7.1 第三種切入角度：教育與親子市場

除了創辦人的「情緒價值」和「資產價值」兩波，有一個被忽略的市場：**親子教育**。

**觀察：**
- 台灣的家長市場（0-12 歲孩子的父母）是一個付費意願高、社群活躍、口碑傳播快的族群
- 「睡前故事」情境可以重新定位為「用你寫的故事，做一本只屬於你孩子的有聲書」
- 家長願意為孩子付費的意願遠高於為自己付費
- 媽媽社群（如 BabyHome、媽咪拜、各種親子 FB 社團）的傳播力極強

**具體場景：**
- 媽媽寫一段孩子的名字嵌入的床邊故事 → 用 VocalCanvas 製作 → 孩子聽到「有人念我的名字」的驚喜感
- 遠距工作的爸爸錄一段睡前故事給孩子 → 但不用自己的聲音（因為覺得自己聲音不好聽）→ 用 VocalCanvas 選一個溫柔的聲音朗讀

**行銷角度：**
- Slogan：「用你寫的故事，陪孩子入睡。」
- 可以做「母親節 / 父親節 / 兒童節」的限時活動
- 親子市場的 LTV 很高——養成習慣後會用好幾年

**風險提醒：** 此建議目前僅基於市場直覺，缺乏台灣親子教育市場的精確數據佐證。建議在投入資源前，先在 1-2 個親子社團做小規模測試（發一篇文看反應），驗證需求是否真實存在。

### 7.2 創辦人可能忽略的行銷機會

#### 機會 1：SEO 免費流量入口

台灣每月搜尋「文字轉語音」「TTS 線上」「AI 配音」的流量估計在數千到萬次。目前沒有任何繁中 TTS 產品認真做 SEO。

**建議：** 做一個免費的「中文文字轉語音」工具頁面，不需要註冊就能轉換 100 字以內的短文。這個頁面的目的不是賺錢，而是佔領搜尋結果第一頁。用戶試了免費版，覺得好用 → 註冊 → 付費。

**成本：** 幾乎為零（頁面開發 + Google Cloud TTS 的微量 API 成本）。

#### 機會 2：「聲音賀卡」的病毒式傳播

做一個「聲音賀卡」功能：
1. 用戶寫一段祝福語
2. 選情境和 BGM
3. 產出一個可分享的連結（像 Spotify Wrapped 那樣的分享頁面）
4. 收件人打開連結 → 聽到一段帶有配樂的祝福
5. 收件人覺得有趣 → 自己也想做一張 → 註冊

這是一個天然的病毒式傳播機制。每一張聲音賀卡都是一次免費的品牌曝光。特別適合在節日（農曆新年、情人節、母親節）做限時活動。

#### 機會 3：與既有療癒品牌的跨界合作

台灣有一些具有知名度的療癒 / 身心靈品牌（如肯園芳療、心靈工坊出版社、各大瑜珈品牌）。與他們合作的價值不在於直接導流，而在於**品牌背書**。

例如：「VocalCanvas x 肯園：一段森林裡的呼吸練習」— 用肯園的文案 + VocalCanvas 的聲音 + 森林系 BGM，做一段聯名冥想音檔。雙方各自在社群推廣，觸及彼此的受眾。

### 7.3 市場分析師的反對意見——行銷端的回應

市場分析報告建議「放下情緒價值優先的執念，先定位為最快的中文配音工具」。這個觀點在商業邏輯上是合理的，但行銷端有不同的考量：

| 分析師觀點 | 行銷端回應 |
|-----------|-----------|
| 情緒價值付費率低 | 同意。但第一波的目標不是付費轉換，是品牌建立。用情緒訴求建立品牌記憶度，比「最快的配音工具」更容易傳播。 |
| 短影音創作者是最大 TA | 同意這是最大市場。但「短影音配音工具」的定位讓 VocalCanvas 直接跟 CapCut 內建 TTS、剪映、ElevenLabs 正面對撞。情緒場景是一個他們不會來打的側翼。 |
| 冥想創作者規模太小 | 部分同意。所以我們的 Persona 不只有冥想老師（Persona B），還有療癒圖文創作者（Persona A）和焦慮上班族（Persona C），後兩者的基數大得多。 |

**結論：兩波策略不是二選一，而是先側翼再正面。** 用情緒價值建立品牌差異化和忠誠度，再用資產價值擴大市場規模。如果直接從「配音工具」切入，VocalCanvas 會淹沒在一堆同質化的競品中。
